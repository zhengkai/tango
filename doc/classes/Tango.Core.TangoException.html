TangoException

Extends \Exception

异常

跟原始 Exception 的区别在于有“深度”的概念，以便更及时的判断错误内容 例如 MySQL 报 sql 语句为空，我们更想知道是哪里发送空 sql， 这个位置信息其实是在 trace 的第三行而非第一行
